## ğŸš€ Getting Started

### Prerequisites

- [.NET 10.0 SDK](https://dotnet.microsoft.com/download/dotnet/10.0) or later
- Any C# IDE (Visual Studio, VS Code, Rider)

### Clone the Repository

```bash
git clone https://github.com/AbanoubPhelopos/cracking-coding-interview-book-solution.git
cd cracking-coding-interview-book-solution
```

### Build and Run

```bash
# Build all solutions
dotnet build

# Run tests
dotnet test

# Run a specific solution
dotnet run --project Chapter01_ArraysAndStrings/Problem01_IsUnique
```
